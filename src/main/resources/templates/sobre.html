Somos estudantes do CTeSP "Tecnologias e Programação de Sistemas de Informação" da Escola Superior de Gestão e Tecnologia de Santarém, este projeto consiste em criar uma base de dados de Pokémons onde seja possivel consultar todos os Pokémons ou utilizar filtos para os consultar, até mesmo, podemos alterar qualquer tipo de dado e podemos apagar um Pokémon.
